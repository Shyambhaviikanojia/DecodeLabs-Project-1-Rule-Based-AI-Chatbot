## 🧠 About the Project

**Ctrl + Alt + Rest** is a simple **Rule-Based AI Chatbot** developed using Python.

The chatbot takes input from the user, identifies specific commands or keywords, and responds according to predefined rules.

It demonstrates the basic concept of rule-based artificial intelligence, where a system follows a set of conditions to decide how it should respond.

The project was created as **Project 1** during my **DecodeLabs Artificial Intelligence Internship**.

## ⚙️ How Does It Work?

The chatbot follows a simple process:

**You type a message**  
↓  
**The chatbot reads your input**  
↓  
**It checks for a matching command or keyword**  
↓  
**The corresponding rule is triggered**  
↓  
**The chatbot gives a response**

For example:

**You:** I'm feeling stressed 😟

The chatbot detects the word **"stress"** and responds with supportive advice.

Some features, such as motivation, study tips, and daily challenges, use Python's **Random Module** to provide different responses.
